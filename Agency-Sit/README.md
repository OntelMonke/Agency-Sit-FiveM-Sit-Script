## ✨ Features

- 🪑 **Automatic Chair Detection** - Detects sittable props automatically
- 🎯 **Smart Positioning** - Automatically finds best seating angle
- 🎨 **Smooth Animations** - Quick and natural stand-up animations
- 🎮 **Ox Target Support** - Works with ox_target if available
- 📢 **Agency-Notify Integration** - Beautiful notifications (optional)
- 🔧 **Framework Support** - ESX, QBCore, or Standalone
- ⚙️ **Highly Configurable** - Customize everything in config.lua

## 📦 Installation

1. Download and extract `Agency-Sit` to your `resources` folder
2. Add to your `server.cfg`:
```
ensure Agency-Sit
```
3. **(Optional)** Install [Agency-Notify](https://agency-script.tebex.io/package/6937769) for better notifications
4. Restart your server

## ⚙️ Configuration

Edit `shared/config.lua` to customize:

- **Agency-Notify Integration** - Enable/disable beautiful notifications
- **Ox Target** - Enable/disable ox_target integration
- **Interact Key** - Change interaction key (default: E)
- **Distances** - Adjust interaction and draw distances
- **Sittable Models** - Add/remove chair models

## 🎮 Usage

- Walk near a chair and press **E** to sit
- Press **E** again to stand up
- With ox_target: Use third-eye interaction
